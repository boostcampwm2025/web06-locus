## 📍 What is Locus?

Locus는 사용자가 특정 장소에서 떠올린 생각·감정·사진 등을 빠르게 기록하고,  
이 기록들이 **지도 기반으로 누적·연결·시각화**되도록 설계된 **위치 기반 기억·사고 구조화 서비스**입니다.

기존 메모 앱, 사진 앱, 노션/옵시디언 등은 기록이 여러 플랫폼에 분산되어 있어  
‘어디에서 어떤 생각을 했는지’, ‘개인의 생활 패턴이 무엇인지’를 파악하기 어렵습니다.  

Locus는 이 문제를 **하나의 기준 — <br>
공간(위치)** 으로 통합하여 기록·경험·사고 흐름을 한 지도에서 확인할 수 있게 합니다.

---

## ✦ 주요 특징

- **위치 기반 통합 기록**  
  텍스트, 사진, 태그 등을 한 번에 입력하고 위치와 함께 저장합니다.

- **지도 기반 시각화**  
  기록을 점(Point)으로만 보여주지 않고,  
  기록 간의 **흐름(선)**, **패턴(클러스터)**, **발생 주기** 등이 시각적으로 드러납니다.

- **즉시 기록 가능한 간단한 UI**  
  별도의 페이지 생성 과정 없이, 도착한 위치에서 바로 기록을 남길 수 있습니다.

- **장소 기반 회고 기능**  
  과거에 기록을 남긴 장소에 다시 방문하면  
  이전 기록을 자동으로 보여주어, 자연스러운 회고 경험을 제공합니다.

---

## 🚀 프로토타입 기능

### 위치 기반 기록 생성

- 현재 위치 자동 인식
- 텍스트·사진 입력
- 태그 설정
- 기록 저장

### 지도 기반 기록 표시

- 기록 점 표시
- 기록 간 거리·연결 기반 시각적 흐름 표현(라인 또는 패턴)
- 다량의 기록을 클러스터링하여 인지적으로 정리된 화면 제공

### 기록 조회 및 회고 (P2 – 프로토타입 범위 외)

> 본 기능은 Locus의 핵심 경험 중 하나이나, 현재 MVP에서는 구현 범위에 포함되지 않습니다. <br>
차후 버전(P2)에서 단계적으로 지원할 예정입니다.

## 💡 사용 시나리오

### 기록 생성

1. 사용자가 특정 장소에 도착
2. 앱에서 "여기에서 기록을 남기겠습니까?" 메시지
3. 텍스트/사진을 빠르게 입력
4. 태그 선택
5. 저장 → 지도에 기록 생성

### 기록 축적 후 탐색

1. 지도 화면에서 기록 분포 확인
2. 특정 구역을 확대하면 클러스터가 풀리고 개별 기록 표시
3. 기록 간 이동 경로 또는 발생 패턴 시각화
4. 사용자는 자신의 사고·경험 흐름 파악 가능

### 회고 기능

1. 사용자가 과거 기록이 있는 장소에 다시 방문
2. "이 장소에서 남긴 기록이 있습니다" 알림
3. 과거 텍스트·사진·감정 등을 확인
4. 새로운 생각을 이어서 기록 가능
